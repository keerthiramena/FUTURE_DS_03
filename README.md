# 📊 Marketing Funnel & Conversion Performance Analysis
**Future Interns | Data Science & Analytics | Task 3**

## 🔍 Overview
Analysis of Bank Marketing campaign data to identify conversion 
drop-offs, channel performance, and opportunities to improve 
lead-to-customer conversion.

## 📁 Dataset
- **Source:** Bank Marketing Dataset (UCI ML Repository)
- **Records:** 45,211 leads | 17 columns

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab | Tableau

## 📈 Key Insights
- **Overall Conversion Rate:** 11.7% (5,289 out of 45,211 leads)
- **Total Drop-off Rate:** 88.3%
- **Best Channel:** Cellular (14%+ conversion rate)
- **Best Month:** March (50%+ conversion rate!)
- **Best Job Segment:** Students (28%+ conversion rate)
- **Worst Channel:** Unknown contact type (4% only)

## 💡 Recommendations
1. Focus campaigns on cellular channel — highest conversion
2. Increase campaign intensity in March, September & December
3. Target student and retired segments — highest converters
4. Avoid unknown contact methods — low ROI
5. Re-engage leads that showed interest (7.4%) but didn't convert
